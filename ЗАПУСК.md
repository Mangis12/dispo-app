# Как запустить проект

## 1. Установить Node.js (один раз)
Скачать с https://nodejs.org → выбрать LTS версию → установить

## 2. Запустить проект
Открыть терминал в папке `dispo-app` и выполнить:

```bash
npm install
npm run dev
```

Сайт откроется по адресу: http://localhost:5173

## 3. Подключить Supabase (опционально)
1. Создать аккаунт на https://supabase.com
2. Создать новый проект
3. Скопировать файл `.env.example` → `.env`
4. Вставить URL и ключ из настроек Supabase
5. Без Supabase — данные хранятся в localStorage браузера

## Структура проекта
```
dispo-app/
├── src/
│   ├── App.tsx          ← Основной компонент
│   ├── types.ts         ← TypeScript типы
│   ├── lib/supabase.ts  ← Supabase клиент
│   ├── index.css        ← Стили (Tailwind)
│   └── main.tsx         ← Точка входа
├── .env.example         ← Пример переменных окружения
├── package.json
└── vite.config.ts
```
